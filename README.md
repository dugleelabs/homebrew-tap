# Duglee Labs Homebrew Tap

Homebrew formulae for Duglee Labs CLI tools.

## Usage

```bash
brew tap dugleelabs/tap
brew install copair
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `copair` | Model-agnostic AI coding agent for the terminal |

## Updating

```bash
brew update
brew upgrade copair
```
